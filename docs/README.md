# Documentation

Public documentation for **HealthSave Observatory** lives here.

This folder is intentionally light for now — user-facing guides (install, capture
setup, the Observatory dashboard, Body Briefs, the private API/CLI, routing, and the
privacy / egress model) are being written and will land here.

In the meantime, start with:

- the repository [`README.md`](../README.md) — overview, quickstart, and capture sources;
- [`API_REFERENCE.md`](../API_REFERENCE.md) — payload-level API reference;
- [`LICENSING.md`](../LICENSING.md) — how the project is licensed.

> Internal planning notes, architecture decision records, and design docs are kept
> out of the public repository.
